# 禅道 Skills

> ⚠️ 注意：当前技能还处于实验阶段，使用时需谨慎！！强烈建议对 AI 调用 API 进行审核。

## 介绍

该项目包括[禅道 API 2.0](https://www.zentao.net/book/api/2309.html) 使用技能 `skills/zentao-api/`，可以获取禅道数据并进行操作。

## 验证方式

* 优先从缓存文件 `～/.zentao-token.json` 获取禅道服务器地址和 Token 信息
* 如果缓存文件没有想过信息，则从环境变量从环境变量获取服务器和 Token 信息
* 如果都无法获取 Token，则从环境变量获取账号密码动态获取 Token 进行使用
* 如果缓存文件和环境变量信息都不存在则提示给出禅道服务地址、账号和密码进行获取
* 获取到 Token 后，会在缓存文件 `～/.zentao-token.json` 中存储，以便下次使用时直接从缓存文件获取 Token 信息

## 注意事项

* ⚠️ 当前技能还处于实验阶段，使用时需谨慎！！强烈建议对 AI 调用 API 进行审核。
* 禅道 Token 永久有效，如果原始 Token 失效，可以要求 AI 清除缓存
* 目前不支持所有模块的接口，后续禅道 API 2.0 支持后会逐步支持所有模块的接口。
